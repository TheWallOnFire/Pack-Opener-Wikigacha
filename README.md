# Pack-Opener-Wikigacha
Quick code for auto open WikiGacha packs
## Getting Started

### Prerequisites
- An active session or account on WikiGacha.

### Usage
1. Open the WikiGacha website.
2. Open the browser's developer console (F12 or Ctrl+Shift+I).
3. Paste the code into the console and press Enter to run.
